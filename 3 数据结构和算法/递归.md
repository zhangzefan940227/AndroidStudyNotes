# 递归

# 递归函数设计的三个重要部分


1. 重要：给**递归函数**一个明确的语义
2. 实现边界条件时的程序逻辑
3. 假设递归函数调用返回结果是正确的，实现本层函数逻辑



# 示例【吃桃】
已知有n个桃子，每天吃一半多一个桃子，最后一天剩余1个桃子，求n

按照三步骤设计递归逻辑：

1. f(n) 表示能吃n天的桃子
2. 当n==1时，桃子数量=1
3. 求f(n)，可以借助n-1天的桃子数量进行计算
    1. n-1天的桃子数量为f(n-1)
    2. f(n) = (f(n-1)+1)*2 

根据这个逻辑，其实思路就非常清晰了，代码如下：

```cpp
int f(int n) {
    if (n==1) return 1;
    return (f(n-1)+1)*2;
}
```
